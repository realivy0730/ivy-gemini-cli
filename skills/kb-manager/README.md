# Knowledge Base Manager

統一的知識庫管理工具,整合檔案整理、去重、標準化、變更記錄。

## 安裝

已安裝於: `~/.kiro/skills/kb-manager/`

## 快速開始

### 完整工作流
```
請使用 kb-manager 整理當前目錄
```

### 獨立操作
```
請使用 kb-manager classify 分類檔案
請使用 kb-manager dedupe 檢測重複
請使用 kb-manager standardize 標準化格式
請使用 kb-manager changelog 生成變更記錄
```

## 功能

- **檔案分類**: 整理至抗幻覺目錄結構
- **重複檢測**: 識別並合併重複內容
- **標準化**: 套用 Markdown 與 Frontmatter 規範
- **變更記錄**: 生成結構化 changelog
